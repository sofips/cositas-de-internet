# Tutorial para crear tu propio jardín

**Estado:** 🌿 brote

Cuando empecé mi jardín quería jugar mucho y pensar poco en el código, por eso está creado desde cero usando Copilot y otras formas de IA.

 El problema fue después cuando quise alentar a más gente a prenderse en esto y no podía explicar muy bien cómo armar sus propios jardines. Si bien entiendo el funcionamiento general, en algún momento  me perdí en la búsqueda del maximalismo. 

 Por eso cree [este template]{https://github.com/sofips/template-jardin-digital} para que cualquiera pueda empezar el suyo. 

 El repositorio incluye un README para que puedas empezar tu jardín de forma relativamente fácil.
