# Tutorial para crear tu propio jardín

**Estado:** 🌿 brote

Cuando empecé mi jardín quería jugar mucho y pensar poco en el código, por eso está creado desde cero usando Copilot y otras formas de IA.

 El problema fue después cuando quise alentar a más gente a prenderse en esto y no podía explicar muy bien cómo armar sus propios jardines. Si bien entiendo el funcionamiento general, en algún momento  me perdí en la búsqueda del maximalismo. 

 Por eso cree [este template]{https://github.com/sofips/template-jardin-digital} para que cualquiera pueda empezar el suyo. 

 El repositorio incluye un README para que puedas empezar tu jardín de forma relativamente fácil.

DISCLAIMER: este template lo hice con un modelo de lenguaje y puede fallar. 

 ### Otras opciones con menos técnica

 - **Tiddlywiki**: Tiddlywiki está muy bueno porque: a) es Open Source , b) se puede usar sabiendo muy poco de código. 

 Se estructura alrededor de *tiddlers* que estructura con links, tags y macros. 

 Además, en combinación con [Tiddlyhost]{https://tiddlyhost.com/sites} permite compartir tu wiki en segundos. 

Aunque no sea tan fácil de personalizar, podés tener tu jardín andando en segundos.

- **Obsidian**: Tiene mucho sentido usar Obsidian porque está pensado para funcionar en una estructura topológica. 

Compartir una *vault* de Obsidian para ser accesible públicamente, no es tan directo pero puede hacerse usando herramientas gratuitas como netlify o vercel.
